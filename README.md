# studentPhotoCollection

#### 介绍
学生照片信息采集

#### 软件架构



#### 安装教程

1.  安装即用
2.  安装前可以先阅读安装说明.txt

#### 使用说明

1.  学生照片信息采集器，支持USB摄像头、笔记本摄像头拍照;
2.  人脸自动识别框选，照片可自动裁剪;
3.  上传Excel自定义数据模板，可选择任意工作表作为数据源;
4.  可自定义选择照片保存路径和照片名;
5.  每次可以检查拍照结果允许重新拍照，可预览照片，采集完成后可导出数据到Excel。
